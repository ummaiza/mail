
# Profil Tri

Template sdh ada Login



## Installation

To deploy this project run
```bash
  git clone https://github.com/ummaiza/template-laravel.git
```

```bash
  composer install
```
```bash
  php artisan migrate --seed
```

Selanjutnya jalankan
```bash
  php artisan serve
```
```bash
  127.0.0.1:8000/login
```
